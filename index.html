Responsive Slideshow with Static Images Beyond Breaking Point

A responsive slideshow with Jquery's cycle2 plugin and bootstrap that falls back to displaying static images on resize beneath breaking point.

Dependencies:

  • JQuery
  • Bootstrap 3
  • Cycle 2

Notes:
Cycle is triggered programatically.

There seems to be a weird bug in cycle 2 that requires in some instances to first trigger 'destroy' before initalizing.

slideshow gets visible after content loaded.
